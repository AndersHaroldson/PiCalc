# PiCalc Prototype

A simple and early version of the eventual PiCalc.

## Current Version Hardware:
* Matrix 4x4 Keypad
* 16x2 LCD Display
* Raspberry Pi 4B
      

## Future PiCalc Hardware:
* Custom PCB
* Indivdual buttons
* Mini LCD display
* Raspberry Pi Zero W 
* Lithium Ion Battery
